# StudyForge - AI Text Humanizer

Paste AI-generated text and humanize it to bypass detection. Features:

- Natural language imperfections
- Grammatical quirks
- Sentence burstiness variation
- Personal touches & anecdotes
- Transitional phrases
- Typo simulation
- Adjustable intensity slider
- Copy & download output

## Usage

Open `index.html` in any browser.

## Tech

HTML, CSS, Vanilla JavaScript — no dependencies.
